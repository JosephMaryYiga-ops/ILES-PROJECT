Internship Logging & Evaluation System User stories 


Student User Stories
1. As a student, I want to log my daily or weekly tasks, so that I can track my work progress.
2. As a student, I want to attach supporting documents or files to my task entries, so that my
workplace supervisor can verify my work.
3. As a student, I want to view my task history, so that I can reflect on my performance over time.
4. As a student, I want to receive feedback from my workplace and academic supervisors on
submitted tasks, so that I can improve my skills.
5. As a student, I want to receive notifications for pending evaluations or deadlines, so that I don't
miss important submissions.

Workplace Supervisor User Stories
1. As a workplace supervisor, I want to view all student logs, so that I can monitor progress.
2. As a workplace supervisor, I want to provide comments or feedback on individual tasks, so that
students receive guidance.
3. As a workplace supervisor, I want to evaluate students periodically using scores or ratings, so that
I can document performance.
4. As a workplace supervisor, I want to generate reports on student performance for the academic
supervisor, so that evaluations are coordinated.

Academic Supervisor User Stories
1. As an academic supervisor, I want to review student logs and workplace evaluations, so that I can
assess academic learning outcomes.
2. As an academic supervisor, I want to provide feedback on student performance, so that they
improve academically.
3. As an academic supervisor, I want to generate academic evaluation reports, so that grades and
assessments can be recorded.

Administrator User Stories
1. As an administrator, I want to create and manage student, workplace supervisor, and academic
supervisor accounts, so that all users can access the system.
2. As an administrator, I want to assign supervisors to students, so that each student has proper
mentorship.
3. As an administrator, I want to generate overall system reports, so that internship and academic
trends are analyzed.
4. As an administrator, I want to set evaluation periods and deadlines, so that students and
supervisors stay aligned with schedules.
